# User Flow Maps

## 1) Upload → Analysis → Decision

1. Upload asset
2. Analysis run starts
3. Decision summary + rationale
4. Recommended actions
5. Export deliverables

## 2) Project → Compare Versions

1. Open project detail
2. Select version A / B
3. Read summary + risk changes
4. Choose winning path

## 3) KPI Review

1. View KPIs
2. Inspect confidence + alignment
3. Decide to iterate or export

# CineView AI Brand System

## Positioning

Primary statement: **CineView AI — Creative Intelligence for Visual Decisions**

## Brand DNA

- Confident
- Calm
- Minimal
- Cinematic
- Premium / Luxury Tech

## Tone

- Director-level
- Strategic
- Never instructional
- Never technical in public UI

## Visual Principles

- Typography is the primary interface.
- No icons, badges, gamified UI, or loud dashboards.
- Contrast is achieved through light-on-dark typography and spatial rhythm.
- Surfaces are subtle; depth is quiet.

## Color System (Cinematic Neutrals)

Use a dark neutral base, soft off-white text, and one restrained accent used rarely.

- Base: `#0B0B0C`
- Carbon: `#111214`
- Ash: `#1A1B1E`
- Mist: `#5C5F66`
- Fog: `#8B9097`
- Ivory: `#EDEFF2`
- White: `#F7F8FA`

### Usage Guidance

- Backgrounds: Base / Carbon / Ash
- Text: Ivory / White, Mist for secondary
- Borders: low-contrast white at 8–12% opacity

## Typography

Display typography should feel editorial and cinematic. Body type should be calm and precise.

### Display

- Family: Neue Montreal (preferred), Satoshi, Inter Tight
- Size: 48–72px
- Line-height: 110–120%
- Weight: 600–700
- Usage: hero, section titles, key statements

### Body

- Family: Inter, IBM Plex Sans
- Size: 14–18px
- Line-height: 150–170%
- Weight: 400–500
- Usage: paragraphs, supporting text, labels

### Micro / Eyebrow

- Size: 10–12px
- Uppercase, tracking +0.3em
- Usage: section markers, meta labels

## Spacing & Rhythm

- Base unit: 8px
- Major section spacing: 48–96px
- Use generous vertical breathing space

## Motion

- Subtle only: fade + minimal translate
- 150–250ms, ease-out
- No heavy animation libraries

## UI Rules (Non‑negotiable)

- No icons
- No badges
- No bright colors
- No gamified UI
- No dashboard noise

## Design Tokens Mapping

Tokens are defined in `app/globals.css`:

- `--cv-base`, `--cv-carbon`, `--cv-ash`, `--cv-mist`, `--cv-fog`, `--cv-ivory`, `--cv-white`
- `--cv-border`
- `--cv-radius`

These tokens are canonical and must be preserved.
